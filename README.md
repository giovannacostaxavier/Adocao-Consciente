# Adoção Consciente – Projeto Web com HTML, CSS e JavaScript

Este projeto é uma página web desenvolvida com foco em praticar e consolidar conhecimentos de HTML5, CSS3 e JavaScript, simulando uma plataforma de adoção consciente de animais.
O objetivo é aplicar conceitos de estrutura semântica, estilização avançada, layout responsivo, integração com serviços externos e interatividade real em um projeto visual e funcional.

## 🌐 Demonstração

🔗 Acesse o projeto online:
👉 [Adoção Consciente](https://projeto-faculdade-ashen.vercel.app/)

## ✨ Sobre o Projeto

O Adoção Consciente é uma plataforma fictícia voltada à adoção responsável de animais, que apresenta:

- Página inicial com apresentação do propósito da plataforma e chamada para ação
- Catálogo de animais disponíveis para adoção, com fotos e informações
- Carrossel de imagens com suporte a swipe (arrastar), para melhor experiência em dispositivos móveis
- Integração com Google Sheets funcionando como banco de dados dinâmico e atualizável
- Hospedagem de imagens via Google Drive, sem necessidade de backend próprio
- Depoimento real de adoção, dando autenticidade à proposta do projeto
- Navegação clara e intuitiva via menu de navegação fixo
- Layout responsivo pensado para diferentes tamanhos de tela
- Design acolhedor com foco em experiência do usuário (UX)

## 🛠 Tecnologias Utilizadas

| Tecnologia | Finalidade |
|---|---|
| HTML5 | Estrutura semântica das páginas |
| CSS3 | Estilização, layout, animações e responsividade |
| JavaScript | Interatividade, manipulação do DOM e consumo de dados |
| Google Sheets API | Banco de dados dinâmico e de fácil atualização |
| Google Drive | Hospedagem das imagens dos animais |
| Flexbox / Grid | Organização e alinhamento dos elementos |
| Vercel | Hospedagem e deploy do projeto |

## 🧩 Funcionalidades

### 🔝 Navegação
- Menu de navegação com links para as seções principais do site
- Navegação suave entre seções da página (scroll interno)

### 🐾 Catálogo de Animais
- Listagem dos animais disponíveis para adoção, alimentada diretamente por uma planilha do Google Sheets
- Atualização de dados sem necessidade de alterar código, apenas editando a planilha
- Imagens hospedadas no Google Drive, carregadas dinamicamente

### 🎠 Carrossel Interativo
- Navegação entre imagens dos animais com suporte a swipe (toque e arraste), otimizado para uso em celular
- Experiência fluida tanto em desktop quanto em dispositivos móveis

## 🎨 Conceitos Trabalhados

- Estrutura semântica com tags HTML5: `header`, `nav`, `main`, `section`, `footer`
- Estilização avançada com CSS3: cores, sombras, bordas, espaçamentos
- Layout com Flexbox/Grid para alinhamento e distribuição de elementos
- Responsividade com unidades relativas e media queries
- Manipulação do DOM com JavaScript para renderizar dados dinâmicos
- Consumo de dados externos via integração com Google Sheets, simulando um banco de dados real
- Uso do Google Drive como CDN improvisada para hospedagem de imagens
- Boas práticas de organização de código e separação de responsabilidades (HTML / CSS / JavaScript)
- UX/UI com foco em clareza visual, hierarquia de informações e apelo emocional
- Deploy com Vercel para publicação gratuita do projeto online

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido para:

- Praticar a criação de layouts de sites reais com HTML e CSS
- Aplicar JavaScript para consumir e manipular dados de uma fonte externa dinâmica
- Explorar soluções alternativas de banco de dados e hospedagem de imagens sem backend próprio
- Trabalhar com carrosséis interativos e suporte a gestos touch
- Organizar arquivos e pastas seguindo boas práticas de projetos web
- Consolidar o entendimento de estrutura, estilo e comportamento no front-end
- Criar um portfólio com projetos visuais e funcionais para apresentar a recrutadores
